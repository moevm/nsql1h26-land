### Веса скоринга

| Фактор | Вес | Описание |
|--------|-----|----------|
| `infra` | 0.35 | Близость к метро, больнице, школе |
| `negative` | 0.25 | Удалённость от свалок, промзон, КАД |
| `features` | 0.25 | Семантические фичи из описания |
| `price` | 0.15 | Нормализованная цена за сотку (ниже = лучше) |

## Требования

- Python 3.11+
- ~600 MB на модель (`paraphrase-multilingual-MiniLM-L12-v2`, скачивается автоматически)
- Jina API ключ (для этапа Reranker)

## Локальный запуск

### 1. Установка зависимостей

```bash
python -m venv .venv
# Windows:
.venv\Scripts\activate
# Linux/macOS:
source .venv/bin/activate

pip install -r avito/requirements.txt
```

### 2. Структура данных

Положите JSON-файл со списком объявлений:

```
data/
└── avito_20260301_140233.json   # входные данные
```

Формат каждой записи в JSON:

```json
{
  "id": 7547791865,
  "title": "Участок 4 сот. (СНТ, ДНП)",
  "description": "Продается участок 4 сотки...",
  "price": 1400000,
  "lat": 59.837,
  "lng": 29.884,
  "locationName": "Петергоф",
  "addressFull": "Ленинградская область, Ломоносовский район...",
  "geoReferences": "д. Низино, СНТ Сад-2",
  "url": "https://www.avito.ru/...",
  "normalizedPrice": null
}
```

### 3. Запуск

```bash
cd avito

# Минимальный запуск — топ-20 по score без текстового запроса
python pipeline.py

# С текстовым запросом (BM25 + Jina Reranker)
python pipeline.py --query "тихий участок с домом и газом рядом с метро" --top 10

# Только BM25 без Jina API (--skip-jina)
python pipeline.py --query "ИЖС лес 10 соток" --top 5 --skip-jina

# С фильтрами по цене и площади
python pipeline.py --min-price 1000000 --max-price 5000000 --min-area 8 --max-area 20

# По конкретному району
python pipeline.py --location "Петергоф" --top 10

# Пересчитать всё (игнорировать кэш)
python pipeline.py --no-cache

# Свой JSON-файл с данными
python pipeline.py --data ../data/test_gas.json --skip-jina --top 3
```

### Все аргументы

| Аргумент | По умолчанию | Описание |
|----------|-------------|----------|
| `--data` | `data/avito_*.json` | Путь к JSON-файлу с объявлениями |
| `--query`, `-q` | — | Текстовый запрос для BM25 + Jina Reranker |
| `--top`, `-n` | `20` | Количество результатов |
| `--min-price` | — | Минимальная цена, ₽ |
| `--max-price` | — | Максимальная цена, ₽ |
| `--min-area` | — | Минимальная площадь, соток |
| `--max-area` | — | Максимальная площадь, соток |
| `--location` | — | Фильтр по названию района/города |
| `--skip-jina` | `false` | Пропустить Jina Reranker (только BM25) |
| `--no-cache` | `false` | Пересчитать всё, игнорировать кэш |

---

## Кэширование

После первого запуска результаты обогащения (фичи + гео) сохраняются в `data/enriched_cache.json`.  
Повторные запуски загружают кэш и пропускают этапы 2–3 (~150 сек экономии).

Сброс кэша: удалите файл или используйте `--no-cache`.

## Docker

### Сборка образа

```bash
# Из корня репозитория
docker build -t avito-pipeline -f avito/Dockerfile .
```

Первая сборка занимает ~5–10 минут: загружается модель (~471 MB) и зависимости Python.

### Запуск контейнера

```bash
# Базовый запуск (данные монтируются через volume)
docker run --rm \
  -v "$(pwd)/data:/app/data" \
  avito-pipeline

# С текстовым запросом
docker run --rm \
  -v "$(pwd)/data:/app/data" \
  avito-pipeline \
  --query "ИЖС с газом и домом" --top 10

# Сохранение кэша между запусками (volume для data/)
docker run --rm \
  -v avito-data:/app/data \
  avito-pipeline --skip-jina --top 5

# Windows PowerShell (замените $(pwd) на ${PWD})
docker run --rm `
  -v "${PWD}/data:/app/data" `
  avito-pipeline --query "тихий участок лес" --top 5
```

### Переменные окружения

| Переменная | Описание |
|-----------|----------|
| `JINA_API_KEY` | Jina AI API ключ (переопределяет значение из config.py) |
| `HF_HUB_DISABLE_SYMLINKS_WARNING` | `1` — убирает предупреждение HuggingFace на Windows |

```bash
docker run --rm \
  -v "$(pwd)/data:/app/data" \
  -e JINA_API_KEY=your_key_here \
  avito-pipeline --query "дом с газом"
```

## Структура проекта

```
avito/
├── pipeline.py          # Главный скрипт — точка входа
├── config.py            # Конфигурация: веса, координаты, модели
├── data_loader.py       # Загрузка и нормализация JSON
├── feature_extractor.py # Семантические фичи (sentence-transformers)
├── geo_scoring.py       # Геоскоринг через geopy
├── scoring.py           # Взвешенный total_score
├── bm25_ranker.py       # BM25 pre-ranking
├── jina_reranker.py     # Jina Reranker API
├── requirements.txt     # Python-зависимости
└── Dockerfile           # Контейнеризация

data/
├── avito_*.json         # Входные данные (объявления)
└── enriched_cache.json  # Кэш обогащённых данных (генерируется)
```
